# Ejercicio de la lectura de Dominando el DOM con JS

En este repositorio se encuentra la solución al reto la lectura de Dominando el DOM con JS donde se debe haacer un generador de contraseñas
Ejercicio: Crear un generador de contraseñas donde: 

- El sitio debe ser capaza de generar una contraseña diferente de mas de 8 palabras usando simbolos, letras y numéros

### Solución 

La solución a este desafio se encuentra en el archivo app.js,index.html y styles.css donde se implementó el maquetado de la página, los estilos y finalmente la lógica para poder crear distintas contraseñas.
