# Bécalos Tech Challenge - Módulo 3: JavaScript

Este repositorio está dedicado a recopilar todos los ejercicios y proyectos desarrollados durante el **Módulo 3** del programa **Bécalos Tech Challenge**. El enfoque principal de esta etapa es el dominio de JavaScript, desde sus fundamentos hasta la manipulación del DOM.

## 📂 Organización del Repositorio

Para facilitar la navegación y el aprendizaje, el repositorio se organiza bajo la siguiente estructura:

* **Carpetas:** Cada tema o proyecto cuenta con su propio directorio independiente.
* **Archivos:** Dentro de cada carpeta se encuentran los archivos correspondientes de cada solución según sea el caso (`.html`, `.css`, `.js`).
* **Documentación Local:** Cada ejercicio incluye su propio archivo `README.md` detallando los objetivos y posibles soluciones.
