# Ejercicio de la lectura de Funciones en JavaScript

En este repositorio se encuentra la solución al reto de de la lectura de funciones en JavaScript donde se resuelve la siguiente problematica.
Ejercicio: Seguimiento de Libros
Crea un sistema muy sencillo para hacer seguimiento de los libros que has leído. (Se baso en libros de programación)
- Define una función `agregarLibro(titulo)`, que añada un libro a un array llamado `librosLeidos`.
- Define una función `mostrarLibrosLeidos()`, que imprima todos los libros que has leído.

### Solución 

La solución a este desafio se encuentra en el archivo `funciones.js`, donde se implementó la solución.
